# Contributing to Ouroboros

Thank you for your interest in contributing to Ouroboros.

## Current Status

This repository is a pre-registration and technical declaration.  
Core development is ongoing in a private environment.

## Contribution Policy

- External contributions are not accepted at this stage.
- Feedback and discussion are welcome via Issues.
- Once the core implementation is published, contribution guidelines will be updated.

## License

All contributions will be under Apache 2.0.
